ξ\xiξ 2.3 逆矩阵

引

 对于数aaa，若有bbb使得ab=ba=1ab = ba = 1ab=ba=1，则称bbb是aaa的倒数，记为a−1a^{-1}a−1或1a\frac{1}{a}a1​.

定义

 1. 对n阶方阵AAA， 若存在方阵BBB使得AB=BA=EAB = BA = EAB=BA=E则称BBB为AAA的逆矩阵，即AAA可逆.
 2. 方阵AAA的伴随矩阵A∗A^*A∗.

续引

  唯一性？存在性？怎么求？

命题

 1. 若方阵AAA可逆，则其逆矩阵唯一，于是记为A−1A^{-1}A−1.
  证：假设AB=BA=EAB = BA = EAB=BA=E，且AC=CA=EAC = CA = EAC=CA=E，
    则B=BE=B(AC)=(BA)C=EC=CB = BE = B(AC) = (BA)C = EC = CB=BE=B(AC)=(BA)C=EC=C.

 2. AA∗=A∗A=∣A∣EAA^* = A^*A = |A| EAA∗=A∗A=∣A∣E.

 3. AAA、BBB是n阶方阵，则AB=EAB = EAB=E（或BA=EBA = EBA=E）⇔a−1=B\Leftrightarrow a^{-1} = B⇔a−1=B.
  证：显然成立
    ∵\because∵ AB=EAB = EAB=E
    ∴\therefore∴ ∣AB∣=∣E∣|AB| = |E|∣AB∣=∣E∣即∣A∣∣B∣=1|A||B| = 1∣A∣∣B∣=1即∣A∣≠0|A| \not= 0∣A∣​=0
    ∴\therefore∴ A−1A^{-1}A−1存在，且A−1=A−1E=A−1(AB)=(AA−1)B=EB=BA^{-1} = A^{-1}E = A^{-1}(AB) = (AA^{-1})B = EB = BA−1=A−1E=A−1(AB)=(AA−1)B=EB=B

 4. 设AAA、BBB是n阶可逆方阵，则
  （1）∣A−1∣=∣A−1∣|A^{-1}| = |A^{-1}|∣A−1∣=∣A−1∣
  （2）(A−1)−1=A(A^{-1})^{-1} = A(A−1)−1=A  (A−1)τ=(Aτ)−1(A^{-1})^{\tau} = (A^{\tau})^{-1}(A−1)τ=(Aτ)−1  (A∗)−1=(A−1)∗=∣A∣−1A(A^*)^{-1} = (A^{-1})^* = |A|^{-1}A(A∗)−1=(A−1)∗=∣A∣−1A
  （3）(λA)−1=λ−1 ⁣A−1(\lambda A)^{-1} = {\lambda}^{-1} \!{A}^{-1}(λA)−1=λ−1A−1（数λ≠0\lambda \not= 0λ​=0）  (AB)−1=B−1 ⁣A−1(AB)^{-1} = {B}^{-1} \!{A}^{-1}(AB)−1=B−1A−1

定理

 方阵AAA可逆⇔∣A∣≠0\Leftrightarrow |A| \not= 0⇔∣A∣​=0，且若AAA可逆，则A−1=A∗∣A∣A^{-1} = \frac{A^*}{\mid A \mid}A−1=∣A∣A∗​.
  证：∵\because∵ AAA可逆 ∴\therefore∴ ∃B\exist B∃B使得AB=EAB = EAB=E
    于是∣AB∣=∣E∣|AB| = |E|∣AB∣=∣E∣，即∣A∣∣B∣=1|A||B| = 1∣A∣∣B∣=1 ，则∣A∣≠0|A| \not= 0∣A∣​=0.
    ∵\because∵ ∣A∣≠0|A| \not= 0∣A∣​=0
    ∴\therefore∴ A(1∣A∣A∗)=(1∣A∣A∗)A=EA(\frac{1}{\mid A \mid}A^*) = (\frac{1}{\mid A \mid}A^*) A = EA(∣A∣1​A∗)=(∣A∣1​A∗)A=E
    ∴\therefore∴ AAA可逆，且A−1=1∣A∣A∗A^{-1} = \frac{1}{\mid A \mid}A^*A−1=∣A∣1​A∗
